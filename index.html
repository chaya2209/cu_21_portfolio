MY PROJECTS

Automatic Water Dispenser

This project is a hardware based project. This project was completed by using Arduino Uno, pH sensor, relay Module, IR proximity sensor, LED, mini-water pump along with jumper wires and battery. The coding part of this project was done on the Arduino IDE. Moreover, the data collected from the sensor was plotted by using PLX-DAQ.


The main purpose of this project was to check whether the water in the container of the dispenser is fit for drinking or not.


Air Quality Monitoring System

This is also a hardware-based project. This project was done by using MQ-2, MQ-7, MQ-135, DHT22 Temperature and Humidity Sensor, Raspberry Pi, and Arduino nano 33 IoT. Arduino IDE was used for the coding part and ThingSpeak Cloud, an analytic data platform was used for making visualizations based on the data collected from the Sensor used.


The main purpose of this project was to check to the quality of air around us.


Smart Water Tank- Web Based Application

This was a team-based project on MEAN stack. In this project, we created a stimulation for a Smart Water Tank. An alert message was shown to the user, whenever the tank was full or about to be empty
